# Jeremy Lutat Portfolio

This is my personal portfolio site built with [Jekyll](https://jekyllrb.com/) using the [Minimal Mistakes theme](https://mmistakes.github.io/minimal-mistakes/), deployed via GitHub Pages.

## 📂 Sections

- **Blog:** Short insights and analytics takeaways from ongoing projects
- **Case Studies:** In-depth explorations of data analytics, product strategy, and brand development
- **Author Sidebar:** Social links open in new tabs

## 🚀 Live Site

View the site at: [https://jeremylutat.github.io](https://jeremylutat.github.io)

## 🛠️ Tech Stack

- Jekyll (v3.10)
- GitHub Pages
- Markdown
- YAML
- HTML/CSS
- FontAwesome
